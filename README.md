# Usage


## Install package

```
pip install threads_sdk
```


## Environment variables

Input those variables in your `.env` file or OS environment (or using `export`).

Then you don't need to input any parameters in `initialize step`.

```
USER_ID
ACCESS_TOKEN
APP_SECRET
```
